# GabyRoos.com - Version 2

* [Hugo](https://gohugo.io/) - install via Brew `brew install hugo`

## Running locally/development

```bash
hugo run server
```

## Running in Docker

```bash
docker build -t gabyroosdotcom:latest .
docker run -p 8020:80 gabyroosdotcom:latest
```
